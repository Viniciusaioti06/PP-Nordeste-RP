# Release 3.0 — Experiência visual

Esta versão preserva a integração existente com Supabase e concentra as alterações na interface.

## Principais mudanças

- Nova camada visual para páginas públicas e painel.
- Microinterações em botões, cards, tabelas, navegação e formulários.
- Animações com respeito à preferência `prefers-reduced-motion`.
- Seção de contato com o Discord oficial.
- Modal pós-inscrição com protocolo, cópia rápida e orientação sobre o Discord.
- Modal de parabéns para candidatos aprovados, com orientação sobre o teste físico.
- Refinamentos de responsividade e hierarquia visual.
- Nenhuma alteração em tabelas, RLS, Edge Functions ou credenciais do Supabase.
